## How to get this project running

Steps:

1. Install project dependencies
    `npm install`
2. Duplicate .env.example as .env
    `cp .env.example .env`
3. Set .env variable
    example: `BASE_HREF=/taiwanbus_taichung`
4. Run locally or build and serve
    `npm run start:dev`
    or
    `npm run start:build`
